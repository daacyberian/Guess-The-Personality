# 🎯 Guess The Personality <img src="https://uploads.mudspike.com/optimized/3X/8/3/8362fa17c2d69eb4c6f11eba355087a383462875_2_457x750.png" width="30" /> - VIP Edition

## ✨ Features

- 🎮 **Interactive Gameplay** - Answer yes/no questions to guess the personality
- 👑 **VIP Dark Theme** - Premium gold and dark theme with smooth animations
- 📱 **Responsive Design** - Works on desktop and mobile
- ⚡ **Fast & Smooth** - Built with React + Vite

## 🛠️ Tech Stack

- **React** - UI Framework
- **Vite** - Build Tool
- **CSS3** - Styling with animations

## 💻 Installation

```bash
# Clone the repository
git clone https://github.com/daacyberian/Guess-The-Personality.git

# Navigate to client folder
cd Guess-The-Personality/client

# Install dependencies
npm install

# Run development server
npm run dev
```

## 🎯 How to Play

1. Think of a famous personality
2. Answer Yes or No to the questions
3. The game will guess who you're thinking of!
